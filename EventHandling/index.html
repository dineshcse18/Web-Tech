EVENT HANDLING

INTRODUCTION TO EVENT HANDLING FUNCTIONS:

  • A function or method containing program statements that are executed in response to an event.

  • An event handler typically is a software routine that processes actions such as keystrokes and mouse movements.

  • With Web sites, event handlers make Web content dynamic.

  • JavaScript is a common method of scripting event handlers for Web content.

  • Events are normally used in combination with functions, and the function will not be executed before the event occurs (such as when a user clicks a button).

EVENT HANDLERS:

  1. onload

  2. onclick

  3. onsubmit

  4. onmouseover

  5. onmouseout

  6. onmousemove

  7. onblur

  8. onsearch

  9. onfocus

  1. ONLOAD:

    • The onload event occurs when an object has been loaded.

    • onload is most often used within the element to execute a script once a web page has completely loaded all content (including images, script files, CSS files, etc.).

    EXAMPLE:

    <html>
    <head>
    <title> load </title>
    </head>
    < body onload="addition()">
    < center >
    < h2 style="color:blue">Event Handling-load()</h2>
    < script type="text/javascript">
    function addition(a,b)
    {
    var a=20;
    var b=30;
    alert("addition performed");
    var c=a+b;
    document.writeln(" < h3 >")
    document.writeln("the addition of a and b is:",c);
    document.writeln("< /h3 >");
    }
    < /script > </center >
    < /body >
    </html>

  2. ONCLICK:

    • The “onclick” event occurs when the user clicks on an element.

    • It runs a specified line of code when you click an HTML object that has the “onclick” attribute.

    EXAMPLE:

    <html>
    <head>
    <title>click</title>
    </head>
    <body>
    <center><p id="a"></p>
    <h2>ON CLICK</h2>
    <button onclick="clik()">Click</button></center>
    <script>
    function clik()
    {
    document.getElementById("a").innerHTML="Hello World!";
    }
    </script>
    </body>
    </html>

  3. ONMOUSEOVER:

    • The onmouseover event occurs when the mouse pointer is moved onto an element, or onto one of its children.

  4. ONMOUSEOUT:

    • The onmouseout event occurs when the mouse pointer is moved out of an element, or out of one of its children.

    EXAMPLE:(MOUSEOVER AND MOUSEOUT)

    <html>
    <head>
    <title>MouseOver</title>
    <style type="text/css">
    h3
    {
    color:orange;
    }
    </style>
    </head>
    <body>
    <center>
    <img src="images.jpg" onmouseover="expand(this)" onmouseout="extract(this)" width="200" height="200">
    <h3>MouseOver</h4>
    <h3>MouseOut</h4></center>
    <script type="text/javascript">
    function expand(x)
    {
    x.style.width="400";
    x.style.height="400";
    }
    function extract(x)
    {
    x.style.width="300";
    x.style.height="300";
    }
    </script>
    </body>
    </html>

  5. ONMOUSEMOVE:

    • The onmousemove event occurs when the pointer is moving while it is over an element.

    EXAMPLE:

    <html>
    <head>
    <title>MouseOver</title>
    <style type="text/css">
    h3
    {
    color:orange;
    }
    </style>
    </head>
    <body>
    <center>
    <img src="images.jpg" onmousemove="expand(this)" onmouseout="extract(this)" width="200" height="200">
    <h3>MouseOver</h4>
    <h3>MouseOut</h4></center>
    <script type="text/javascript">
    function expand(x)
    {
    x.style.width="400";
    x.style.height="400";
    }
    function extract(x)
    {
    x.style.width="300";
    x.style.height="300";
    }
    </script>
    </body>
    </html>

  6. ONSEARCH:

    • The onsearch event occurs when a user presses the "ENTER" key or clicks the "x" button in an input element with type="search".

    EXAMPLE:

    <html>
    <head>
    <title>onsearch</title>
    <style type="text/css">
    h2
    {
    color:blue;
    }
    h3
    {
    color:red;
    }
    </style>
    </head>
    <body>
    <h2>
    Enter the search object:<input type="search" id="obj" onsearch="srch()"></h2>
    <h3 id="con"></h3>
    <script>
    function srch()
    {
    var z=document.getElementById("obj");
    document.getElementById("con").innerHTML="Your searching object is:"+z.value;
    }
    </script>
    </body>
    </html>

  7. ONBLUR:

    • The onblur event occurs when an object loses focus.

    • The onblur event is most often used with form validation code (e.g. when the user leaves a form field).

  8. ONFOCUS:

    • The onfocus event occurs when an element gets focus.The onfocus event is most often used with <input>, <select>,and <a>.

  9. ONSUBMIT:

    • The onsubmit event occurs when a form is submitted.

    • The onsubmit event is commonly used to validate the contents of form controls and set the contents of some input:hidden elements before submitting. If the user forgot to fill some required fields or made some other mistake on the form, you can cancel the submission with the onsubmit event.

    EXAMPLE:(ONBLUR,ONFOCUS,ONSUBMIT)

    <html>
    <head>
    <title>onblur and onfocus</title>
    </head>
    <body>
    <pre>
    <form onsubmit="sub()">
    FIRST NAME :<input type="text" id="fname" onblur="slice()">
    LAST NAME :<input type="text" id="lname" onfocus="foc(this.id)">
    <h3 id="div"></h3></pre>
    <input type="submit" value="Submit">
    <script>
    function slice()
    {
    var x=document.getElementById("fname");
    var y=x.value.slice(0,4);
    document.getElementById("div").innerHTML="Your sliced name is:"+y;
    }
    function foc(z)
    {
    document.getElementById(z).style.background="yellow";
    }
    function sub()
    {
    alert("submitted successfully");
    }
    </script>
    </body>
    </html>
