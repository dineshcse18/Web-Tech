JavaScript String Object

JavaScript strings are used for storing and manipulating text.

  • Strings are created as primitives: var firstName="John"
  • But strings can also be created as objects using the new keyword: var firstName = new String("John")

    • String Properties
    s.no Property Syntax Description
    1 Constructor string.constructor Returns a reference to the String function that created the object.
    2 Length string.length Returns the length of the string.
    3 Prototype object.prototype.name = value The prototype property allows you to add properties and methods to an object.

    String Method
    S.no Method Syntax Description
    1 charAt() string.charAt(index); Returns the character at the specified index.
    2 charCodeAt() string.charCodeAt(index); Returns a number indicating the Unicode value of the character at the given index.
    3 concat() string.concat(string2, string3[, ..., stringN]); Combines the text of two strings and returns a new string.
    4 indexOf() string.indexOf(searchValue[, fromIndex]) Returns the index within the calling String object of the first occurrence of the specified value, or -1 if not found.
    5 lastIndexOf() string.lastIndexOf(searchValue[, fromIndex]) Returns the index within the calling String object of the last occurrence of the specified value, or -1 if not found.
    6 localCompare() string.localeCompare( param ) Returns a number indicating whether a reference string comes before or after or is the same as the given string in sort order.
    7 match() string.match( param ) Used to match a regular expression against a string.
    8 replace() string.replace(regexp/substr, newSubStr/function[, flags]); Used to find a match between a regular expression and a string, and to replace the matched substring with a new substring.
    9 search() string.search(regexp); Executes the search for a match between a regular expression and a specified string.
    10 slice() string.slice( beginslice [, endSlice] ); Extracts a section of a string and returns a new string.
    11 split() string.split([separator][, limit]); Splits a String object into an array of strings by separating the string into substrings.
    12 substr() string.substr(start[, length]); Returns the characters in a string beginning at the specified location through the specified number of characters.
    13 subString() string.substring(indexA, [indexB])
    		 Returns the characters in a string between two indexes into the string.
    14 toLocalLowerCase() string.toLocaleLowerCase( ) The characters within a string are converted to lower case while respecting the current locale.
    15 toLocalUpperCase() string.toLocaleUpperCase( ) The characters within a string are converted to upper case while respecting the current locale.
    16 toLowerCase() string.toLowerCase( ) Returns the calling string value converted to lower case.
    17 toUpperCase() string.toUpperCase( ) Returns the calling string value converted to uppercase.
    18 toString() string.toString( ) Returns a string representing the specified object.
    19 valueOf() string.valueOf( ) Returns the primitive value of the specified object.

    Javascript Math Object

    • The Math object allows you to perform mathematical tasks.
    • All the properties and methods of Math are static and can be called by using Math as an object without creating it.
    • Math is not a constructor.
    • It provides you properties and methods for mathematical constants and functions.

    Syntax

    The syntax to call the properties and methods of Math are as follows

    Math Object Properties
    S.No Properties Description
    1 E Euler's constant and the base of natural logarithms, approximately 2.718
    2 LN2 Natural logarithm of 2, approximately 0.693
    3 LN10 Natural logarithm of 10, approximately 2.302
    4 LOG2E Base 2 logarithm of E, approximately 1.442
    5 LOG10E Base 10 logarithm of E, approximately 0.434
    6 PI Ratio of the circumference of a circle to its diameter, approximately 3.14159
    7 SQRT1_2 Square root of 1/2; equivalently, 1 over the square root of 2, approximately 0.707
    8 SQRT2 Square root of 2, approximately 1.414

    Javascript Boolean Object

    • The Boolean object represents two values, either "true" or "false".
    • If value parameter is omitted or is 0, -0, null, false, NaN , undefined, or the empty string (""), the object has an initial value of false.

    Syntax

    The syntax to call the properties and methods of Boolean are as follows

    Boolean Object Properties
    S.No Property Syntax Description
    1 constructor boolean.constructor() Returns a reference to the Boolean function that created the object
    2 prototype object.prototype.name = value The prototype property allows you to add properties and methods to an object


    Boolean Object Methods
    S.No Method Syntax Description
    1 toSource() boolean.toSource() Returns a string containing the source of the Boolean object; you can use this string to create an equivalent object
    2 toString() boolean.toString() Returns a string of either "true" or "false" depending upon the value of the object
    3 valueOf() boolean.valueOf() Returns the primitive value of the Boolean object
