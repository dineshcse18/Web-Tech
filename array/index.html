WHAT IS ARRAY?

  • An array is a group of memory locations that all have the same name and normally are of the same type.

  • To refer to a particular location or element in an array,we specify the name of the array and the position number of the particular element in the array.

  • Consider,if 'c' is the array variable, the first element in c is the zeroth element (i.e) c[0], the second element as c[1], and so on.,

  • The position number in square brackets is called subscript

    • *Syntax: var array_name=[item1,item2,....];
    *Example:var fruit=["apple","orange","mango"];

DECLARING AND ALLOCATING ARRAYS

  • Arrays occupy space in memory.

  • 'new'operator is used to allocate the memory dynamically.

  • The process of creating new objects is also known as creating an instance or instantiating an object

  • operator new is known as the dynamic memory allocation operator

    *Syntax: var array_name=new Array(array_size);
    *Example:var c=new Array(12);

ACCESS THE ARRAY ELEMENTS OF AN ARRAY

  • you access an array element by referring to the index number.

    • Example:
    < html >
    < head >
    < title > Index < /title >
    </head>
    <body>
    <script type="text/javascript">
    var fruit=["apple","orange","mango"];
    document.writeln(fruit[0]);
    </script>
    </body>
    </html>

  • you may also change the value of array element.

    • Example:
    <html> <head>
    <title> Index </title >
    </head>
    <body>
    <script type="text/javascript">
    var fruit=["apple","orange","mango"];
    fruit[0]="papaya";
    document.writeln(fruit[0]);
    </script>
    </body>
    </html>

  • The full array can be accessed by referring to the array name.

    • *Example:
    <html>
    <head>
    <title> Index </title >
    </head>
    <body>
    <script type="text/javascript">
    var fruit=["apple","orange","mango"];
    document.writeln(fruit);
    </script>
    </body>
    </html>

  • Arrays are a special type of objects.The typeof operator in JavaScript returns "object" for arrays.

  • Objects use names to access its "members"

    • Example:
    <html>
    <head>
    <title> Index </title >
    </head>
    <body>
    <script type="text/javascript">
    var fruit={firstfruit:"apple",secondfruit:"orange",thirdfruit:"mango"};
    document.writeln(fruit["firstfruit"]);
    </script>
    </body>
    </html>

  • The length property of an array returns the length of an array (the number of array elements).

    • Example:
    <html>
    <head>
    <title> Index </title >
    </head>
    <body>
    <script type="text/javascript">
    var fruit=["apple","orange","mango"];
    document.writeln(fruit.length);
    </script>
    </body>
    </html>

  • You may access the last element by using [array.length-1].

    • Example:
    <html>
    <head>
    <title> Index </title >
    </head>
    <body>
    <script type="text/javascript">
    var fruit=["apple","orange","mango"];
    document.writeln(fruit[fruit.length-1]);
    </script>
    </body>
    </html>

  • you may also use "for" loop

    • Example:
    <html>
    <head>
    <title> Index </title >
    </head>
    <body>
    <script type="text/javascript">
    var fruit=["apple","orange","mango"];
    len=fruit.length;
    for(i=0;i<len;i++)
    document.writeln(fruit[i]);
    </script>
    </body>
    </html>

  • you can also use array.forEach() function

    • Example:
    <html>
    <head>
    <title> Index </title >
    </head>
    <body>
    <script type="text/javascript">
    var fruit=["apple","orange","mango"];
    fruit.forEach(sample);
    function sample(value)
    {
    document.writeln(value);
    }
    </script>
    </body>
    </html>
